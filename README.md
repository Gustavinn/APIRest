# APIREST

  Essa aplicação foi desenvolvida para fins de estudos.
  Api rest, CRUD, deploy no Heroku. 

# Back-End:

## Tecnologias
- Java 8
- Spring Boot 2.7.2
  - spring-boot-starter-web
  - spring-boot-starter-data-jpa
  - spring-boot-devtools
- H2 DataBase
- Maven (Embedded no Spring Boot)
- Swagger2
- Swagger UI
- Git
